# EchoCart Backend Setup

## Prerequisites

Install:

- Docker Desktop

Make sure Docker Desktop is running before continuing.

---

## 1. Clone / Download Backend Setup

Download the Repo using Github Desktop (Check YT)

---

## 2. Start Backend

Open Command Prompt terminal inside the EchoCart folder.

Run:

```bash
docker compose up
```

---

## Backend URL

Once containers are running:

```text
http://localhost:8080
```
